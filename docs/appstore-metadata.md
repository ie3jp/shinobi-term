## Description (EN)

The shortest path from iPhone to Claude Code.

Shinobi Term is a free, open-source SSH terminal client for iOS. Connect to your Mac's tmux session with one tap and start working with Claude Code — from anywhere.

**Why Shinobi Term?**

- One-tap tmux attach: See your active tmux sessions and connect instantly
- Built for Claude Code: Designed for developers who run Claude Code in tmux
- CJK-first: Japanese, Chinese, and Korean characters render correctly
- SSH key auth: Generate Ed25519 keys on-device, private keys stay in Keychain
- Extended keyboard: Ctrl, Alt, Esc, Tab, arrow keys
- Scroll mode: Browse tmux output history with swipe gestures
- Free & open source: No ads, no tracking, no account required

**Recommended Setup**

iPhone + Tailscale + Shinobi Term → Mac's tmux → Claude Code

Install Tailscale on both your Mac and iPhone for seamless SSH access from anywhere. Create a tmux session, start Claude Code, and attach from your iPhone with one tap.

**Privacy**

All credentials are stored in iOS Keychain. No data is sent to our servers. No analytics or tracking.

Source code: https://github.com/IE3/shinobi-term

## Review Notes

Shinobi Term is a developer tool — an SSH terminal client for connecting to remote servers.

To test the app, you need an SSH server to connect to. The app connects to user-specified SSH servers using the standard SSH protocol (port 22). No special accounts or credentials are needed beyond a standard SSH server.

The app does not require any special permissions beyond network access for SSH connections.
